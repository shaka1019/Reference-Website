6.4 Functions with multiple parameters

Notes

Last time we learned to add a parameter to our function definitions. Sometimes we might need more data that just one parameter. Luckily, we can define functions to have as many parameters as we want. To define multiple parameters, we just need to define multiple variable names in the function's () separated by commas.

Function definition with multiple parameters 

function myFunction(param1, param2, param3){
  // Function statements
}
