5.10 Random Numbers

Notes

Intro

Random numbers are useful in software development for many reasons- game mechanics, generating usernames/passwords, anytime you need something seemingly unpredictable.
Pseudo-random-numbers- computers are just big calculators solving formulas and can't create truly randoms numbers, but instead use an algorithm that generates numbers that seem random.

Random Numbers in JS

Javascript provides a number of functions that are useful for generating random numbers.
Math.random() is a function that returns a pseudo-random number between 0 and 1, not incliding 1.
Math.floor() is a function that takes a decimal number and rounds it down-chops off the digital part.

Random Integer Calculations

To produce a random integer from a min value to a max value (including the min and max) use this calculation.

Example

Generate a random number from 1 to 10 amd store it in a variable
var randNum = Math.floor(Math.random() * 11) + 1;
